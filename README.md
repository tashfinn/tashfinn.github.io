# Miftahul Islam Tashfin - Personal Portfolio Website

Welcome to the source code of my personal portfolio website, hosted live at [tashfin5.github.io](https://tashfin5.github.io).

## 🎨 Overview

This is a blazing-fast, ultra-modern portfolio website built completely from scratch using **Vanilla HTML, CSS, and JavaScript**. By avoiding heavy frameworks, the site achieves instant loading times and complete control over every pixel of the design.

### Key Features
- **Premium Dual-Theme Design:** Features a deep charcoal Dark Mode and a crisp, clean Light Mode, easily toggled via the navbar. The preference is saved locally using `localStorage`.
- **Custom Glassmorphism:** Sleek, semi-transparent frosted glass elements used for the project and experience cards.
- **Dynamic Scroll Animations:** Implemented custom Intersection Observers in JavaScript to trigger smooth `fade-in-up` animations as you scroll down the page.
- **Persistent Glowing Background:** CSS-animated gradient orbs float smoothly in the background, anchored to the viewport.
- **Fully Responsive Layout:** Optimized for mobile phones, tablets, and massive desktop displays using CSS Grid and Flexbox.

## 🛠 Tech Stack

- **Structure:** HTML5 (Semantic Structure)
- **Styling:** CSS3 (CSS Variables, Flexbox, Grid, Keyframes)
- **Interactivity:** Vanilla JavaScript (ES6+)
- **Icons:** [Phosphor Icons](https://phosphoricons.com/)
- **Typography:** Google Fonts (`Inter` & `Outfit`)

## 🚀 Running Locally

Because this project uses vanilla web technologies, there are no dependencies to install or complex build steps.

1. Clone the repository:
   ```bash
   git clone https://github.com/tashfin5/tashfin5.github.io.git
   ```
2. Simply open `index.html` in any modern web browser to view the site instantly.
